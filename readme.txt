Afftra - Ultimate affiliate marketing WordPress Gutenberg Blocks.

== Description ==

Afftra - The Ultimate Gutenberg Block Plugin for Affiliate Marketing. 

**Afftra** is a cutting-edge Gutenberg-based WordPress block plugin designed to enhance your affiliate marketing efforts. With its sleek, modern design and powerful features, Afftra allows users to effortlessly showcase their affiliate or digital products from various platforms directly within the Gutenberg editor. The plugin's intuitive drag-and-drop system, combined with extensive customization options, makes it an indispensable tool for any affiliate marketer.

### Key Features of Afftra

* **Simple, Lightweight, and Fast**: Afftra is designed to be efficient, ensuring your website remains fast and responsive.
* **Developed with Gutenberg Native Components**: Built using native Gutenberg components, ensuring seamless integration and compatibility.
* **Drag-and-Drop Functionality**: Easily add and arrange blocks within the Gutenberg editor without coding knowledge.
* **Highly Customizable**: Offers a wide range of customization options to tailor the appearance and functionality of your blocks to match your brand.
* **Clean & Smart Customization Panels**: User-friendly customization panels that make it easy to adjust settings and styles.
* **No jQuery Dependency**: Built without relying on jQuery, ensuring better performance and compatibility with modern web standards.

Afftra is the perfect solution for affiliate marketers looking to create visually appealing and highly functional product showcases within their WordPress sites. Whether you're promoting digital products or physical goods, Afftra's robust features and ease of use make it a must-have plugin for your affiliate marketing toolkit.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/afftra-blocks` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
